Excel Format Guide

Column Required Notes
A: username Required Unique login name. Will be lowercased automatically.
B: fullName Required Student's full name as displayed.
C: email Required Must be unique across all students.
D: phone Optional 10-digit mobile number. Can be left blank.
E: password Optional If blank, defaults to username@123

Tips: Row 1 must be the header row (it's skipped during import). Duplicate usernames or emails are skipped automatically and shown in the summary. Download the template above to get started quickly.
